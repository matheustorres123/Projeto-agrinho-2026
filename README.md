# 🌿 Agrinho 2026 - Campo e Cidade: Conexões que Transformam

## 📌 Objetivo do Projeto

Este projeto foi desenvolvido para o Agrinho 2026 com o tema “Campo e Cidade: Conexões que Transformam”.

O objetivo principal é demonstrar a importância da relação entre o meio rural e urbano, mostrando como o campo e a cidade dependem um do outro para o desenvolvimento econômico, social e ambiental.

Além disso, o projeto busca conscientizar sobre sustentabilidade, tecnologia no campo e os Objetivos de Desenvolvimento Sustentável (ODS), valorizando também a região do Paraná e a cidade de Campo Mourão.

---

## 💻 Tecnologias Utilizadas

- HTML5 (estrutura do site)
- CSS3 (estilização e layout)
- JavaScript (interatividade e animações)

---

## ⚙️ Recursos Técnicos

- Estrutura em seções (single page)
- Menu de navegação com âncoras (#)
- Cards organizados em grid
- Estatísticas animadas com JavaScript
- Quiz interativo com respostas
- Layout responsivo (adaptável ao celular)
- Imagens ilustrativas externas e locais
